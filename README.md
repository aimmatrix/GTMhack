# GTM

Noodle / Lightfern Reach hackathon website.

Noodle finds relevant people or businesses, prepares research packets, and hands that context to Lightfern for email drafting.

## Run locally

```bash
python3 -m http.server 4173
```

Then open:

```text
http://127.0.0.1:4173/
```

## Files

- `index.html` - page structure and copy
- `styles.css` - visual design and responsive layout
- `script.js` - demo interactions for the animated prism, modal, and research packet switching
