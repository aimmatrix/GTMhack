# Animated Demo Slug

This is a short animated browser asset for the coffee-bean supplier demo.

Open it directly in a browser:

```text
demo-slug/index.html
```

Or serve the repo and open:

```bash
python3 -m http.server 4173
```

```text
http://127.0.0.1:4173/demo-slug/
```

Suggested use: screen-record the 12-second animation and place it before the live skit or use it as the product-screen insert.

For exact frame checks, add a time query:

```text
http://127.0.0.1:4173/demo-slug/?t=7.8
```
